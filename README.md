# Single-Cell Pathway Analysis in Proliferative CLL Cells

## Overview

This repository contains code for the analysis of single-cell RNA-seq data from CLL patients.

The study focuses on identifying pathway activity within proliferative tumor cells, with particular emphasis on IFN-α, IFN-γ, and KRAS signaling programs associated with elevated AID expression and disease progression.

## Data

- Chronic lymphocytic leukemia (CLL)
- Single-cell RNA-seq
- 4 patients
- Tumor proliferative fraction

## Analyses

- Cell filtering and normalization
- Tumor cell identification
- Proliferative cell characterization
- Differential expression analysis
- Pathway enrichment analysis
- IFN-α pathway analysis
- IFN-γ pathway analysis
- KRAS pathway analysis
